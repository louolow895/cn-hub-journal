# cn-hub-journal

本仓库用于归档和发布多个独立的 HTML 页面。内容以静态页面形式存放，不针对任何特定域名或具体网站。

## 项目简介

cn-hub-journal 是一个通用型 HTML 页面归档仓库。页面内容独立、结构清晰，适用于需要长期保存或定期更新的静态页面集合。仓库本身不提供搜索、评论或动态功能，所有页面均为纯前端实现。

## 目录说明

```
cn-hub-journal/
├── index.html          # 仓库首页/导航页
├── pages/              # 存放各独立 HTML 页面
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
├── assets/             # 公共资源（CSS/JS/图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/` 目录下存放每个独立页面，按需命名。
- `assets/` 目录存放页面共用的样式、脚本及图片资源。
- 根目录下的 `index.html` 可作为页面索引或导航入口。

## 页面归档说明

- 每个页面均为独立的 HTML 文件，可直接通过浏览器打开。
- 页面间无强依赖关系，可单独查看或部署。
- 归档内容不包含动态后端、数据库或用户交互逻辑。
- 页面样式和脚本尽可能内联或引用 `assets/` 中的公共资源。

## 维护说明

- 如需添加新页面，请在 `pages/` 下创建 HTML 文件，并更新 `index.html` 中的链接（如有）。
- 请保持文件命名规范，避免特殊字符或中文文件名。
- 公共资源如有更新，请同步修改引用路径。
- 本仓库不推荐使用外部 CDN，优先使用本地资源以保证长期可用性。

## 贡献

欢迎提交 Issue 或 Pull Request 来改进页面内容或结构。提交前请确保页面在主流浏览器中可正常显示，并检查无外部资源依赖（除非必要）。
